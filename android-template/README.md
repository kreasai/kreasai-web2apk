# Android Template Scope

Template Android minimal yang akan dibangun:
- Kotlin
- WebView (JavaScript enabled)
- SwipeRefreshLayout
- Runtime permission handler (GPS, Camera, dll)
- External app deep link handling (`whatsapp://`, `mailto:`, `tel:`)
- Offline detection + custom error state
- Back button support untuk history WebView
- Extendable splash screen config

Catatan: implementasi detail template bisa dipindah ke module `app-template/` di repo ini.
